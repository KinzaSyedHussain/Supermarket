# 🛒 Supermarket Drag-and-Drop Game

A fun, interactive 2D supermarket checkout simulation built with **React** and **Vite**. This project mimics a real-world point-of-sale (POS) system where players can select grocery items, scan them at a counter, view totals on a digital monitor, and handle cash transactions using standard bills and coins.

## ✨ Features

* **Interactive Grocery Aisle:** A variety of fast food and dessert items equipped with responsive scaling effects on hover.
* **HTML5 Drag-and-Drop Implementation:** * Drag food items from the shelves directly onto the register counter to scan them.
    * Drag cash bills and coins directly into the register drawer to settle the balance.
* **Dynamic POS Monitor:** Updates in real-time to show the item name scanned, total items, overall bill cost, and calculates accurate change due once payment is completed.
* **Animated Cash Drawer:** Splitting the workflow into a scanning phase and a paying phase keeps the UI clean, dropping down an animated cash wallet when it's time to pay.
